Our paper [SpecRover: Code Intent Extraction via LLMs](https://arxiv.org/abs/2408.02232) has been accepted to ICSE'25. 
SpecRover (aka AutoCodeRover-v2.0) guides an LLM agent with program specifications
inferred from multiple sources and is the best-performing open-source agent on SWE-Bench!
